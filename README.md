# Realtime Chat & Threads App

## Tech Stack
- Frontend: Next.js, Clerk, Tailwind
- Backend: Node.js, Express, Prisma
- Database: PostgreSQL
- Auth: Clerk
- Infra: Docker Compose

## Running Locally
docker-compose up -d
npm run dev (frontend & backend)

## Architecture
- Monorepo with frontend & backend
- Clerk-based authentication
- Prisma ORM
